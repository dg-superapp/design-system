# Label

Form field label built on `@radix-ui/react-label`. Pairs with any control via `htmlFor`. Optional required asterisk renders in `--red-600` (the only `--red-600` usage in Phase 3).

## Installation

<CopyInstallButton name="label" />

## Props

| Prop       | Type      | Default | Description                                                         |
| ---------- | --------- | ------- | ------------------------------------------------------------------- |
| `htmlFor`  | `string`  | —       | The `id` of the associated control. Required for a11y pairing.      |
| `required` | `boolean` | `false` | Appends a `--red-600` asterisk with `aria-hidden="true"`.           |
| `disabled` | `boolean` | `false` | Applies `--muted-foreground` to match the associated disabled input.|

## Variants

Label has two visual states and an optional required decoration.

| State    | Foreground            | Asterisk     |
| -------- | --------------------- | ------------ |
| Default  | `--foreground`        | hidden       |
| Required | `--foreground`        | `--red-600`  |
| Disabled | `--muted-foreground`  | hidden       |

## Usage

### Khmer / ភាសាខ្មែរ

```tsx
import { Label } from "@/components/ui/label";
import { Input } from "@/components/ui/input";

export function EmailFieldKm() {
  return (
    <div className="flex flex-col gap-2">
      <Label htmlFor="email-km" required>
        អាស័យដ្ឋានអ៊ីមែល
      </Label>
      <Input id="email-km" type="email" required placeholder="name@example.com" />
    </div>
  );
}
```

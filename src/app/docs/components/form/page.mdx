# Form

Form bundle built on `react-hook-form` + `zod`. Exports seven canonical sub-parts — `Form`, `FormField`, `FormItem`, `FormLabel`, `FormControl`, `FormMessage`, `FormDescription` — that wire accessible error messaging, `aria-invalid` injection via Radix `Slot`, and a polite `aria-live` region for each field's error text. Error colour is `--danger` (`--red-700`) throughout.

## Props

| Sub-part          | Key props / notes                                                                                     |
| ----------------- | ----------------------------------------------------------------------------------------------------- |
| `Form`            | Spread of `useForm()` return. Wraps children in `FormProvider`.                                        |
| `FormField`       | `name`, `control`, `render`. Binds a controller to one RHF field.                                      |
| `FormItem`        | Vertical stack, `gap: --space-1` (4 px). Generates a stable field id via `useId`.                      |
| `FormLabel`       | Inherits `Label` (§ 2.8). Supports `required` asterisk in `--red-600`.                                 |
| `FormControl`     | Radix `Slot`. Injects `aria-invalid`, `aria-describedby`, and `id` into the wrapped control.           |
| `FormDescription` | `--text-xs`, `--muted-foreground`. Rendered above `FormMessage`, connected via `aria-describedby`.     |
| `FormMessage`     | `--text-xs`, `--danger`. `aria-live="polite"`. Only renders when the field has an error.               |

## Variants

Form exposes no visual variants — it is a behavioural bundle. Error state is derived automatically from RHF's `fieldState.error`. Try the [form playground](/preview/form/) for invalid-state screenshots and focus-on-first-error behaviour.

| State   | FormControl                  | FormMessage                    |
| ------- | ---------------------------- | ------------------------------ |
| Valid   | no `aria-invalid`            | not rendered                   |
| Invalid | `aria-invalid="true"`        | `--danger` text, `aria-live`   |

## Usage

### Khmer / ភាសាខ្មែរ

```tsx
"use client";

import { zodResolver } from "@hookform/resolvers/zod";
import { useForm } from "react-hook-form";
import { z } from "zod";
import {
  Form,
  FormControl,
  FormField,
  FormItem,
  FormLabel,
  FormMessage,
} from "@/components/ui/form";
import { Input } from "@/components/ui/input";
import { Button } from "@/components/ui/button";

const schema = z.object({
  email: z.string().email("អាស័យដ្ឋានអ៊ីមែលមិនត្រឹមត្រូវ"),
});

export function SignInFormKm() {
  const form = useForm<z.infer<typeof schema>>({
    resolver: zodResolver(schema),
    defaultValues: { email: "" },
  });
  return (
    <Form {...form}>
      <form onSubmit={form.handleSubmit((v) => console.log(v))}>
        <FormField
          control={form.control}
          name="email"
          render={({ field }) => (
            <FormItem>
              <FormLabel required>អាស័យដ្ឋានអ៊ីមែល</FormLabel>
              <FormControl>
                <Input type="email" {...field} />
              </FormControl>
              <FormMessage />
            </FormItem>
          )}
        />
        <Button type="submit">បញ្ជូន</Button>
      </form>
    </Form>
  );
}
```
